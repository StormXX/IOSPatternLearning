###ObserverPattern 观察者模式
实现观察者模式有两种方式

- NSNotification
- KVO

用两种方式实现了三个滑块改变rgb值从而控制View的背景颜色。
具体可看我的博客-[IOS设计模式之观察者模式](http://dangu.me "点我点我！")
